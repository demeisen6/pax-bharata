# Pax Bharata

A scenario mod for
[Terra Invicta](https://store.steampowered.com/app/1176470/Terra_Invicta/):
an alternate 2026 Earth of 170 novel nations — a contained Mongol Empire,
a surviving Abbasid Caliphate, a China of a hundred kingdoms, and an age
of colonization driven from the Indian subcontinent instead of Western
Europe.

**Steam Workshop:**
https://steamcommunity.com/sharedfiles/filedetails/?id=3767900104

**The full history of the world:** https://demeisen6.github.io/pax-bharata/

## Manual install

If you prefer not to use the Workshop:

1. Download [`PaxBharata.zip`](PaxBharata.zip) (or grab the raw files from
   [`mod/PaxBharata/`](mod/PaxBharata/)).
2. Extract the `PaxBharata` folder into
   `...\Steam\steamapps\common\Terra Invicta\Mods\Enabled\`.
3. Launch the game and pick the **Pax Bharata** scenario when starting a
   new campaign. Vanilla scenarios remain playable with the mod enabled.

**Known limitation:** unification/fragmentation projects are
scenario-locked but not scenario-hidden — each scenario shows the other's
projects in the tech tree, permanently un-unlockable.

**Supported game version:** built against the current (pre-Dark Skies)
build of Terra Invicta. A compatibility update will follow shortly after
Dark Skies launches.
